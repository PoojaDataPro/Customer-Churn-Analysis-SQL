# Customer Churn Analysis using MySQL

## Project Description
This project focuses on analyzing customer churn data using MySQL.
The goal is to identify churn patterns and understand factors affecting customer retention.

## Tools Used
- MySQL 8.0
- SQL Queries
- GitHub

## Files in this Project
- churn_queries.sql : SQL queries used for analysis
- churn_analysis.txt : Key insights from the analysis
- dataset_info.txt : Information about the dataset

## Key Analysis Performed
- Active vs Churned customer analysis
- Month-wise customer churn analysis
- Subscription end-date trend analysis
- Plan-wise churn analysis
- City-wise customer distribution
- Top customers based on subscription/activity
